# OpAMP Commander

## Components
### opampctl
- opampctl is a command line tool to control opampcommander.

#### Install
```sh
# opampctl
go install github.com/minuk-dev/opampcommander/cmd/opampctl@latest

# create config file
opampctl config init
```
